Program

We offer a variety of programs to suit all levels of experience and fitness.
Our programs are designed to help you achieve your goals, whether you are a beginner or an advanced practitioner.

Beginners Class

This program focus in teaching the basic fundaments of jiu-jitsu and fitness.

Fundamental Class

This program is for students with basic knowledge of jiu-jitsu. They will learn how to trasition to one technique to another. They will start sparring.

Advanced Class

For students with advance knowledge of jiu-jitsu.
This program seeks not only to improve students jiu-jitsu basic skills, but also, teaching advances techniques.

Kids Class

Classes are divided in 2:

  • 4-7 years old (K1)
  • 8-12 years old (K2)
K1 classes are the first step for kids to learn jiu-jitsu at the early age. They will learn the basic of jiu-jitsu and fitness. K2 classes are for kids that already have some knowledge of jiu-jitsu. The class will have the same structure as the adults class.

Self Defence

This program are included on the Basic and the Fundamentals class.
Students will learn self-defence techniques during thoses classes.

Ladies class

This class is mostly jiu-jitsu fitness and very basic techniques.
Only women are allowed in this class.
